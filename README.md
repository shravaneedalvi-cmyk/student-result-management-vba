# Student Result Management System (Excel VBA)

## Overview

This project is built using Excel VBA to manage student marks. It allows users to add student details, calculate total marks, percentage, grades, and view class statistics automatically.

## Features
- Add new student records using VBA Input Boxes
- Automatic total and percentage calculation
- Automatic grade assignment
- Display class average percentage
- Display highest percentage
- User-friendly interface with buttons for data entry and result processing

## Technologies Used

- Microsoft Excel
- Excel VBA (Visual Basic for Applications)

- RawMarks.xlsm – Main Excel VBA project
